숫자 text에 일정 값 더하기(구분자 이용)  

파일명 : addtext1.lsp
이미 기입된 숫자 text을 원하는 값만큼 증가시키거나 감소시키는 기능입니다. addtext와 같은 기능이지만 addtext1은 구분자를 지정할 수 있습니다. 즉 기존 텍스트가 EL=14.234이나 GL-3.456, I.L+3.543 과 같은 형태인 경우 =,-,+를 구분자(delimiter)로 지정해서 구분자 뒤쪽의 수에 대해서 값을 더하거나 빼기를 합니다. 
 사용예
Command: addtext1
Decimail Precision New/<Old>:  	[New: 소수점 자리수 변경을 위해서 N입력, 
									Old:원래 자리수 유지하려면  또는 O입력] 
Enter delimiter(ex:=): =		[구분자 입력]
Enter Add value: 1.5			[증감 값 입력]
All/<Select>:       		[All: 모든text 선택,  원하는 text만 선택]
Select text: w{p1} {p2} 	[Window box를 이용하여 Text선택]
Select objects:       	[선택종료]
2 Modified						[수정된 text개수 출력됨]
