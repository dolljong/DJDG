외곽선폰트의 내부를 Solid hatch로 채우기 
 
  파일명 : txtfil.lsp


  윈도우용 AutoCAD가 나왔을 때 사용자들이 가장 기대하던 것중 하나가 윈도우용 폰트를 쓸 
수 있을까? 하는 것이었습니다. 예상대로 사용할 수는 있었지만 기대에 미치지는 못했습니다. 윈
도우용 폰트인 truetype폰트를 사용한 경우 작업속도가 현저히 떨어지는 현상이 나타났? 것입니
다. 그래서 아직까지도 캐드용 한글프로그램들이 많이 쓰이고 있는 실정입니다. 한글 프로그램들
을 사용하는 이유중의 하나가 바로 안이 채워진 글자를 만들어주는 기능때문일 것입니다. 이프로
그램은 미약하나마 그러한 작업을 해줍니다. 일단 외곽선 폰트를 이용해서 text를 만든 다음 이 
프로그램을 이용하면 안이 채워진 text를 얻을 수 있습니다. 


  
 사용예 
  영문폰트중 대표적인 외곽선 폰트인 outline.shx를 이용하여 solid text를 만들어보겠습니다.
  우선 [Fortmat] [Text style] 메뉴에서 font를 outline.shx를 선택하고 원하는 text를 만든 다음 
txtfil명령을 사용합니다. 주의할 사항은 명령을 사용하기 전에 regen명령으로 원이나 호로 이루어
진 폰트 모양을 매끄럽게 만들어주는 것이 좋습니다. 또한 현재 화면정보를 이용해서 hatch를 하
는 것이므로 text가 화면에 꽉 차도록 zoon in해서 hatch를 해주는 것이 좋습니다.  



Command: txtfil
txtfil
Select text to be FILLED: 
Select objects: {Select s1} 1 found      [채울 text를 선택]
Select objects:                   [선택을 끝내기 위해서 Enter]
