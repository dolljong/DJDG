Draw rounded rebar

 
  File name : RBAR.LSP


  This program changes line type of rounded rebar part to deshed type. 
  It changes part of given R value length.

 
  Example 
    Start point: {p1}                   [Pick start point to select rebars]
    End point: {p2} 7 LINES found               [Pick end point to select rebars]
    Round length: 500                  [Enter Radious of Round. This length is changed to Dashed line]
