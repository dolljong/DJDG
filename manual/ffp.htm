여러번 offset해줍니다.
 
 Program name : FFP.LSP


  R먹은 교량의 슬라브 배근도를 그리다보면 CAD에 있을 것 같은 명령이 없다는 것을 알게됩니
다. 어떤 상황이냐하면, 슬라브 배근도에서 배력근은 R이 계속 변하며 내측부터 외측까지 배근됩
니다. 이것을 쉽게 그릴 수 있는 기능이 CAD에 있을 것 같은데, 없습니다. Array명령은 똑같은 R
을 가진 호를 만들어 주기때문에 이 경우에는 쓸모가 없습니다. 그래서 일일이 offset을 시켜야 
합니다.(이구 지겨워~) 이 프로그램은 그러한 번거로움을 없애줍니다. 두 점을 찍어주면 프로그램
은 두점간의 거리를 구하고 사용자가 입력한 offset거리로 나누어 몇번을 offset할 것인가를 정한
다음 그 횟수만큼 offset해줍니다.  
  


 
 사용예
Pick start point: end of {p1}           [시작점]
Pick end point: end of {p2}             [끝점]
Offset distance<2000.0>: 200   [offset distance]
Select object: {p3}                     [offset될 entity선택]
