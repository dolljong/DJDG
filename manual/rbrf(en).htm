Draw front view of main rebar of circular column .
 
  File name : RBRF.LSP


  This program draws frontview of main rebar of circuler column.

 
  Example 
    Select first line: {p1}                     [Pick first rebar line]
    Select second line: {p2}                    [Pick second rebar line]
    All round/Half round <All round>:   [Choose opetion 'A'll to draw all round. If you want draw half round, choose option 'H'alf]
    Number of Rebar: 70                        [Enter number of rebars. It is all rebar of section]
