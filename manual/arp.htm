호를 따라 array해줍니다.
 
  Program name : ARP.LSP


  예를 들어 R먹은 교량의 슬라브 배근도를 그릴 때 슬라브 주철근을 배근하려면 array명령의 
polar option을 이용하여 array야합니다. 하지만 원하는 C.T.C로 배근하려면 계산기를 몇번 두드려
야 합니다. 이 프로그램을 이용하면 그런 과정 없이 시작점과 끝점을 찍어주어 array polar를 수행
합니다.

 
 사용예
Select Arc or Circle: {p1}              [기준이 될 호/원]
Pick start point: end of {p2}           [array할 시작점]
Pick End point: end of {p3}             [array할 끝점]
Enter Delta: 100                       [array간격, 호의 길이]
Select objects: {p4} 1 found            [array될 entity]
Select objects:                        [선택 끝냄]
