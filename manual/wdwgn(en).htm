Write current file name on drawing
 
  Program name : WDWGN.LSP


   This program will write file name on the drawing. 
  you can pick the location where you want to write the drawing name. 
  The file name is obtained by referring to the AutoCAD environment variable named DWGNAME. 



 
   Example 
  Pick insert point: {Pick the location where you want to write the drawing name}


  
¡Ø Notice
  Height of file name text is 2mm. That is, If scale of drawing is 1:50, Height of file name become 100mm.
