호를 나누어줍니다.


  Program name : DIVA.LSP


  위에서 소개한 MEASA.LSP에서와 마찬가지로 호위의 두점 사이를 원하는 갯수로 나누어 표시해
줍니다. AutoCAD가 제공하는 DIVIDE명령의 응용이라 하겠습니다. 프로그램은 두점의 길이(호의 길
이)를 구하고 그 길이를 사용자가 원하는 길이로 나누어 표시를 해줍니다. 표시는 point명령을 이
용하고, 그 크기나 형태는 현재 설정된 값으로 만들어집니다.

 
 사용예
Select Arc or Circle: {ARC선택}
Pick start point: end of {p1}   [호 위의 한 점을 찍어줌]
Pick end point: int of {p2}     [호 위의 한 점을 찍어줌]
Pick Direction Pint: {p3}     [p1으로 부터 어느 방향(시계방향/반시계방향)으로 divide할 것
                                                 인가를 정해줌]
Number of divide: 5         [몇등분 할 것인가를 정해줌]
