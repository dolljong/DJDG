좌표 list를 만들어줍니다.
 
  Program name : NUMCOOR.LSP


  좌표를 구하고자 하는 점들을 찍어주면 순서대로 번호를 매기고 그 좌표값들을 도면상에 기입
하거나 파일로 만들어줍니다. 선형작업, 슈좌표작업, 모델링작업 등을 할 때 유용하리라 생각합니
다.특히 모델링 작업시 좌표값은 m단위로 구하는 반면, 도면은 mm단위로 그리므로 mm를 m로 
환산할 필요가 있습니다. 이러한 작업을 위해서 Scale factor를 입력할 수 있도록 했습니다.

 
 사용예
Start Number <1>:                      [시작번호=1]
Text hight <150.0>:                    [글씨 크기 지정]
Scale factor <1.0>: 0.001              [구해진 좌표값에 곱해질 숫자]
Pick location of Coordinate list: {p1}  [좌표list가 위치할 곳]
