Draw plan view of main rebar of circular column 
 
  File name : RBRP.LSP


This program help you to determin number of rebars.
If you enter minimum distance between rebars, 
program suggest passible number of rebar and actual distance. 
If you want to change suggested number or distance You can option N or option C
Program iterates until you Enter O.K.

 
 
 

 


     Example 
    Pick Center point: {p1}                     [Pick center of column]
    Post DIA(m): 2.5                           [Enter Diameter of column]
    Number of layer: 2                 [Enter number of rebar layer]
    *** 1 LAYER Cover(mm): 100         [Enter Cover of 1'st layer]
    Number/<C.T.C(mm)>: 100            [Enter C.T.C, If you want enter Number. Enter N option]
    Number of rebar: 72  New C.T.C.(mm): 100.356        
                                                [Program suggest Number and C.T.C]
    change Number/Ctc/<Ok>: n          [Enter N to enter new number]
    Number of Rebar: 70                        [Enter numbr 70]
    Number of rebar: 70.0  New C.T.C.(mm): 103.224 
                                                [Program displays new C.T.C When Number is 70]
    change Number/Ctc/Ok:              [Enter O.K to finish 1st layer ]
    *** 2 LAYER Cover(mm): 200         [Enter cover of 2nd layer (from surface)]
    Number/<C.T.C(mm)>: 100            [Enter C.T.C(100), If you want enter Number. Enter N option]
    Number of rebar: 65  New C.T.C.(mm): 101.498        
                                               [Program suggest Number and C.T.C]
    change Number/Ctc/<Ok>: n          [Enter N to enter new number]
    Number of Rebar: 60                        [Enter numbr 60]
    Number of rebar: 60.0  New C.T.C.(mm): 109.956 
                                                [Program displays new C.T.C When Number is 60]
    change Number/Ctc/Ok:                      [Enter O.K to finish 2nd layer]
