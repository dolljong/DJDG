Draw plan view of rounded embankment
 
   File name : rnori.lsp


  This program draws embankment that is made outside of wing wall. This program assume that original ground is level

 
   Example
Pick Center point: {p1}    [Pick the Top poit of cone.] 
Pick start point: {p2}     [시작각도를 지정합니다. 여기서는 도로의 직각방향]
Pick End point: {p3}       [끝각도를 지정합니다]
첫단 노리길이: {p2}       [첫단 노리의 길이를 입력합니다. 점을 찍어주거나 숫자를 입력해도 
                          됩니다]
2 단 노리길이: 1500  [2단노리의 길이를 입력합니다]
3 단 노리길이:       [더이상의 노리가 없으면 를 입력하면 아래와 같이 그림이 그
                            려집니다]
