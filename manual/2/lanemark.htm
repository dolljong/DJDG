차선표시를 그려줍니다.
 
  파일명 : lanemark.lsp


  차선표시를 하는 것은 간단한 것 같은데 막상 하려면 짜증나는 작업입니다. 교량의 노면은 항
상 기울어져 있기 때문에 Array명령을 사용하려고 해도 차선폭은 수평거리이기 때문에 생각처럼 
쉽게 작업이 가능한 것은 아닙니다.

 
   사용예
  command: lanemark
  Select line: {s1}                     [노면선을 선택한다]
  Pick X-position: {p1}                         [첫번째 marking의 x위치를 찍어준다]
  Pick side: {p2}                               [방향을 찍어준다]
  Enter Number: 2                        [차선수를 입력한다]
  Enter Width of lane: 3600      [차선의 폭을 입력한다]
