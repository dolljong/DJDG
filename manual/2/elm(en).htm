Insert Elevation mark and Write Elevation
 
    Program file name : ELM.LSP


  This program insert \BLOCKS\ELM.DWG file as a block. 
This block is just arrow. Program draws lines. The length of line under text is automaticaly adjusted to the length of text.

 
 Example
Pick insert point: {P1}  [Pick insertion point of arraw head]
Pick Vertical side: {P2} [Pick Upper or lower]
Pick Horizontal side: {P3} [Pick Left or right]
Elevation: 23.456 [Enter Elevation value. Program adds'EL.=']
