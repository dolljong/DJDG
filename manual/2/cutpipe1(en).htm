Draw wave shape cutting mark
 
   Program file name : CUTPIPE1.lsp


  This program is used when you ommit a part of entity. This program draw a ommit line of wave shape, and trim ommited part
 
 Example
Command: cutpipe1
Pick First point: {p1}   [Pick first point of ommit line]
Pick Second point:  {p2}   [Pick second point of ommit line]
Pick Side: {p3}   [Pick side being ommited]
