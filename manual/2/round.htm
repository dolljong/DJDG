ROUND효과를 줍니다.
 
    파일명 : ROUND.LSP


    보통 교각의 원형기둥이나 R먹은(?)코핑부분 등에 쓰이는 기능으로서 곡면을 표시해줍니다. 
전체가 원인 경우(예:원기둥)와 반만 원인경우(예:타원형기둥)를 표현 할 수 있습니다. 아래의 
사용예1과 사용예2를 참조하시길..
 
   사용예-1 (All round)
  Select first line: {P1}  [첫번째 line 선택]
  Select second line: {P2} [두번째 line선택]
  All round<Half round>: A [All round기능 선택,
                            default: Half round] 
  
 
  
 
   사용예-2 (Half round)
  Select first line: {P1}  [첫번째 line 선택]
  Select second line: {P2} [두번째 line선택]
  All round<Half round>:  [default인 Half round기능 선택]
