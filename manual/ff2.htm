중심선을 기준으로 양쪽으로 offset합니다.
 
  Program name : FF2.LSP


  강재도면을 그리다보면 짜증나는(나만 그런가?) 작업중의 하나가 얇은 판을 표현하기 위해서 
center line을 중심으로 양쪽으로 offset시켜주는 작업입니다. plate 두께가 두껍다면 별 문제가 안
되겠지만 대부분의 토목도면에서의 plate두께는 화면상에서 작업하기 귀찮을 만큼 얇습니다. 더군
다나 offset시켜야 할 entity들이 많을 경우에는 zoom in을 해서 center line의 양쪽으로 offset을 시
킨다음 다시 zoom out을 하는 작업을 여러번 반복해야 합니다. 그리고 offset을 시키면 offset전의 
색과 line-type이 그대로 적용됩니다. 그래서 다시 색깔과 line-type을 바꿔주는 작업을 해야합니다. 
이 프로그램은 그러한 불편함을 없애기 위해 태어났습니다(?) 양쪽으로 offset시키고 싶은 entity들
(아직은 line과 arc entity만 가능)을 선택하면 사용자가 원하는 두께만큼 offset을 수행해줍니다.

 
사용예
Offset distance<15>: 10[offset distance 입력 (center에서 offset된 entity까지 거리)
                                                   여기에서 강재두께는 20mm가 됨)]
Color number<7>:              [offset된 line(arc)의 색깔(흰색이 default)]
Continuous/Hidden/CEnter/Dashed<Continuous>:
                                [offset된 line(arc)의 line-type(실선이 default)]
Select object: {p1} {p2}        [cross box를 이용하여 offset시킬 entity들 선택]
Select object:                [선택을 끝내기 위해서 Enter입력]
