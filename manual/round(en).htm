Draw rounded surfece
 
    Program file name : ROUND.LSP


    It is usually used for circular column, Rounded part, and it displays curved surface.
You can express the case where the whole is rounded (e.g., the cylinder) and the case where the half is rounded  
See example 1 and example 2 below.
 
   Example-1 (All round)
  Select first line: {P1}  [Select first line]
  Select second line: {P2} [Select second line]
  All round<Half round>: A [Select All round, default: Half round] 
  
 
  
 
   Example-2 (Half round)
  Select first line: {P1}  [Select first line]
  Select second line: {P2} [Select second line]
  All round<Half round>:  [Select Half round(default)]
