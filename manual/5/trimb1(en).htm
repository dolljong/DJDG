trim between lines.
 
  File name : TRIMB1.LSP


There are many case that lines are intersect in steel drawing, because steel plate has thickness.
In this case, We have to trim lines that is cutted plate.
At that time, We have to zoom in to trim lines. 
It is kinda picky. you just do select two line. 
Program trims every entities between lines that you select.
 
 Example
select first entity: {p1}       [Select first line that is not cutted]
select first entity: {p2}       [Select second line that is not cutted]
