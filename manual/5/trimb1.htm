두께를 가진 강재의 겹치는 부분을 정리해줍니다.
 
  Program name : TRIMB1.LSP


  강재 도면을 그리다 보면 어떤 두께를 가지는 강재들이 교차하는 경우가 있습니다. 이렇게 교
차하는 부분의 어느 한쪽 강재는 잘라져야하는데, 이러한 작업을 CAD로 하려면 일단 zoom in으
로 선들이 교차하는 부분을 확대한 후 trim명령으로 교차하는 부분의 선들을 잘라줍니다. 아시다
시피 CAD에서 zoom in, zoom out은 시간을 소요하는 명령입니다. 그리고 trim명령으로 선들을 선
택하는 불편함을 없애기 위해서, 이 프로그램에선 단지 두 선만 선택해주면 그 사이의 모든 선들
을 trim하여줍니다. 주의할 점은 두 선 사이에 있는 모든 entity들이 trim된다는 것입니다. 선택의 
기회는 없습니다. (물론 잘못?을 댄 undo를 하면 되겠죠..)

 
 사용예
select first entity: {p1}       [잘리지 않는 강재선 선택]
select first entity: {p2}       [잘리지 않는 강재선 선택]
