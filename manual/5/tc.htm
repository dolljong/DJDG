강재두께변화를 표시해줍니다.
 
  Program name : TC.LSP


  강재 도면에서 강판의 두께가 변하는 부분에서 두꺼운 쪽을 1 : 5의 기울기로 그라인딩 한후 
용접합니다. 이 부분을 표현해주는 프로그램입니다. 곡선헌치부의 플랜지나 R을 먹은 곡선 교량의 
WEB은 아래 그림 같이 원형입니다. 이러한 부분도 처리해줍니다.  



 
   사용예
  Select thin plate:    [얇은 강판을 선택합니다. 아래 그림에서 오른쪽 위 ARC]
  Select thick plate:   [두꺼운 강판을 선택합니다. 아래 그림에서 왼쪽 위 ARC]
