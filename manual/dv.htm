수직치수선을 그려줍니다.
 
  Program name : DV.LSP


AutoCAD의 치수선 명령중 VER명령가 같은 기능을 가지고 있으며, 앞에서 소개한 DH와 같이 연속
치수선, 골뱅이(@) 등의 추가기능을 지원합니다.

 
 사용예
Pick first point/Object: {p1}   [치수선의 시작점]
Pick Dimension line side: {p2}  [치수가 쓰여질 위치 좌/우 (여기서는 오른쪽)]
Level <1>:                             [치수선 단계 (여기서는 1단)]
Distance or RETURN to Pick point: 2000 [알고있는 거리는 직접 입력함]
Dimension text <2.000>:                [만일 text를 고치고 싶으면 여기서 바꿀 수 있음]
Distance or RETURN to Pick point/eXit:       [거리를 모른다면 점을 찍어 입력하기 위해 Enter]
Pick point: end
of {p3}                                 [한점을 찍어줌, 프로그램은 점의 y값만 인식함]
Dimension text <16.200>: 16.200(10.000)        [새로운 text를 입력함]
Distance or RETURN to Pick point/eXit:       [거리를 모른다면 점을 찍어 입력하기 위해 ]
Pick point: end {p4}                    [한점을 찍어줌, 프로그램은 y값만 인식함]
Dimension text <750>:          [text를 그대로 사용]
Distance or RETURN to Pick point/eXit:       [거리를 모른다면 점을 찍어 입력하기 위해 ]
Pick point: end {p5}                            [한점을 찍어줌, 프로그램은 y값만 인식함]
Dimension text <1.102>:                        [text를 그대로 사용]
Distance or RETURN to Pick point/eXit: x       [DV명령 끝냄]
