Rotate reference
 
  파일명 : rref.lsp


  어떤 물체를 다른 어떤 선의 각도와 같도록 돌리려면 rotate명령중 reference 옵션을 사용하는
데 새로운 각도(즉 원하는 각도)를 회전의 중심점에서부터 측정하도록 되어있기 때문에 불편합니
다. 이 프로그램을 사용하면 원하는 각도를 두점으로 지정할 수 있습니다. 

 
   사용예
Command: rref
Select objects: {select s1}    [회전시킬 물체를 선택합니다]
Select objects: 1 found
Select objects:         [선택을 끝냅니다]
Base point: {pick p1}          [회전 중심선 찍어줌]
Reference angle: {pick p1}  Specify second point: {pick p2} [기준이 되는 각도 지정]
New angle: {pick p3} Specify second point: end of {pick p4} [회전시킬 각도를 지정]
