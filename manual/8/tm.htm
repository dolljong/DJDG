기존의 글자에 원이나 box로 테두리를 씌워줌. (글자에 모양내기?)
 
  Program name : TM.LSP


  이미 쓰여진 글자에 마킹을 해줍니다. 마킹의 종류는 원, box, 타원, 오방떡(?) 모양등 4가지가 
있습니다. box, 타원, 오방떡 option을 쓰면 text크기에 따라 자동으로 마킹의 크기를 정할 수 있는 
"Fit" option이 있습니다. "Fit" option을 쓰지 않으면 마킹의 크기를 물어 봅니다.

 
 사용예-1 원
Select objects: Other corner: 1 found                   [모양을 낼 text선택]
Select objects:   Circle/Box/Ellipse/Obang: c [원모양을 선택]
Pick center point:                                      [원의 중심]
Radius:                                         [반지름]

 
 사용예-2 BOX
Select objects: 1 found                                 [모양을 낼 text선택]
Select objects:   Circle/Box/Ellipse/Obang: b [box모양 선택]
<Box width>/Fit:  Second point:                 [두점을 찍어서 box폭 입력, 숫자입력 가능]
Box Height:  Second point:                      [두점을 찍어서 box높이 입력, 숫자입력 가능]

 
 사용예-3 타원
Select objects: 1 found                         [모양을 낼 text선택]
Select objects:  Circle/Box/Ellipse/Obang: e   [타원형 선택]
<Box width>/Fit:  Second point:         [두점을 찍어서 타원폭 입력, 숫자입력 가능]
Box Height:  Second point:                      [두점을 찍어서 타원높이 입력, 숫자입력가능]

 
 사용예-4 트랙(오방떡)
Select objects: 1 found                                 [모양을 낼 text선택]
Select objects:  Circle/Box/Ellipse/Obang: o           [타원형 선택]
<Box width>/Fit:  Second point:            [두점을 찍어서 타원폭 입력, 숫자입력 가능]

Box Height:  Second point:                 [두점을 찍어서 타원높이 입력, 숫자입력가능]
