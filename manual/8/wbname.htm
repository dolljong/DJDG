Wblock의 file명을 text로부터...
 
 
  파일명 : wbname.lsp

  보통 도면작업을 할 때 한파일에 한 장의 도면을 그리는 것이 아니고 한파일 안에 여러장의 도
면을 그리는 경우가 많습니다. 하지만 CD납품을 할 때는 이렇게 한 파일에 여러장의 도면을 넣
어두어서는 안됩니다. 검색하기가 힘들기 때문이지요. 그래서 작업할 때는 여러장을 한 파일에서 
작업을 하지만 나중에 납품할 때는 따로 따로 잘라내는 작업을 해야합니다. 보통 wblock명령으로 
그러한 작업을 하게 되는데 일일이 파일이름을 입력하기 불편하고 오타의 가능성이 있기 때문입
니다. 이 프로그램은 파일이름을 도면에 있는 텍스트를 이용해서 정할 수 있기 때문에 도면에 있
는 도면번호나 도면명등을 선택하여 파일이름을 정할 수 있습니다. 


  
 사용예 
Command: wbname
Select objects: {Pick s1} 1 found  [파일이름이 될 text를 선택함]
Select objects:  
                                       [선택을 끝내기 위해 Enter침, 여러개를 선택하면 계속해서 더해짐]
Insertion base point: {Pick p1}  [삽입점 찍어줌]
Select objects: {Pick p1} Specify opposite corner: {Pick P2} 5 found 
                                     [block으로 만들 개체를 선택함]
