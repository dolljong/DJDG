Write coordinate.


  Program name : COORD.LSP, COORD2.LSP, COORD3.LSP


  When doing alignmen work or calculating shoe coordinates, all coordinates are calculated manually before CAD is introduced.
But We don't have to calculate it every time I work with CAD. That is, We draw the structure following alignment. 
and We can get the coordinates on the CAD by reading coordinates. This program writes the coordinates of CAD to the drawing.

 
 Example
Pick First Point: {p1}  [Pick the point that You want to write the coordinate]
Pick Second Point: {p2} [Pick the point that You want to write the coordinate]
