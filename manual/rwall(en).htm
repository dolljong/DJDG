Draw retaining wall section using Dialog box.
 
  File name : DDRWALL.LSP


  You can enter dimensions of retaining wall by using dialog box. Insert point is top front of the wall.
  You can select a data file of rwall.exe made by Bae mun kyo(PE)
   Example 
  commad: ddrwall
  {Click Ok button after enter dimensions of retaining wall}
  insert point: {p1}    [Pick tip front of retaining wall]
