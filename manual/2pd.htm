두점간의 거리를 도면에 기입해줍니다.


  Program name : 2PD.LSP


  도면작업에서는 별로 없는 일이지만 모델링 작업등에서 가끔 두점간의 거리를 직접 도면에 기
입해야 할 때가 있습니다. 

 
 사용예
Text height: 150                       [text의 크기지정]
Text angle: 0                  [text의 회전각]
Prefix: L=                             [두점간의 거리를 도면에 입력할 때의 머릿말]
Pick first point: {p1}          [첫점]
Pick second point: {p2}         [두번째점]
Pick Text point : {p3}          [text가 쓰여질 위치]
Pick first point:                      [명령을 끝내기 위해 Enter, 계속 반복할 수 있슴]
