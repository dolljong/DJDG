Divide a part of Arc.


  Program name : DIVA.LSP


  This is similar to MEASA.LSP. This program divide a part of arc.
Program get distance distance between two points that user picks, and divide that part.
points are added on divided points. See MEASA.

 
 Example
Select Arc or Circle: {Select ARC}
Pick start point: end of {p1}   [Pick a start point on arc]
Pick end point: int of {p2}     [Pick a end point on arc]
Pick Direction Pint: {p3}     
                         [Pick direction point. That is, from p1 to clckwise or counterclockwise]
Number of divide: 5         [Ente divide number]
