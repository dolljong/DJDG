Array polar between seclected two point
 
  File name : ARP.LSP


  For example, to draw slab reinforcement of an curved bridge , you must array 
using the polar option of the array command. But you have to use the calculator to get to the CTC(arc length). 
With this program, you can do array from start point to end point without using calculater. 

 
 Example
Select Arc or Circle: {p1}              [Select a arc or circle]
Pick start point: end of {p2}           [Pick start point of array]
Pick End point: end of {p3}             [Pick end point of array]
Enter Delta: 100                       [Enter array distance, It is length of arc]
Select objects: {p4} 1 found            [Select a entity that is done arrat]
Select objects:                        [Type enter to finish selecting]
