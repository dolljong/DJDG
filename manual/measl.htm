선을 measure해줍니다.

 
  Program name : MEASL.LSP


  MEASA.LSP에서와 마찬가지의 기능을 가지고 있으나 LINE에 대해서 measure를 수행합니다. 
measure된 점의 표시는 point와 block이 가능합니다. 
  


 
 사용예-1
Pick start point: end of {p1}           [시작점]
Pick end point: nea to {p2}             [끝점]
<Segment length>/Block: 500    [measure길이]
Total length= 2458.16           [시작점과 끝점사이의 거리를 표시해줌]
Number of segment= 4                    [measure길이의 갯수를 표시해줌]

 
 
 

 

 
 사용예-2
Pick start point: end of {p1}           [시작점]
Pick end point: int of {p2}             [끝점]
<Segment length/Block: b               [block으로 표시]
Block name: star                       [block의 이름]
Align block with object? <Y>: y        [block을 시작점과 끝점의 각만큼 돌린것인가?]
Segment length: 500                    [measure길이]
Total length= 2323.77           [시작점과 끝점사이의 거리를 표시해줌]
Number of segment= 4                    [measure길이의 갯수를 표시해줌]
