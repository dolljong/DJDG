슈표시를 해줍니다.
 
      파일명 : SHOE.LSP


  슈좌표도를 그릴때 슈를 표시해줍니다. 체인이 직선인 경우엔 작업이 용이하지만 원인 경우에
는 원의 접선방향이나 법선방향으로 슈마킹을 회전시켜야합니다. 이러한 작업을 쉽게 하기위해 
만들어졌습니다. 슈마킹은 아래의 3개 block을 insert합니다.
·\BLOCKS\SHOEMF.DWG (고정단)
  \BLOCKS\SHOEM1.DWG (1방향 가동단)
  \BLOCKS\SHOEM2.DWG (2방향 가동단) 



   사용예
  Select Chain:                 [여기서 chain을 선택합니다. 직선이나 원, 호만 인식합니다.]
  Pick shoe insert point :      [슈의 삽입점을 찍어줍니다.]
  MOVE Fix/Longi-/Trans-/2-dir: [ Fix    : 고정단
                                       Longi- : 교축방향 가동
                                       Trans- : 교축직각방향 가동
                                        2-dir  : 양방향 가동단]
