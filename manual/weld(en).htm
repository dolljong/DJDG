Draw welding symbol.
 
  Program name : WELD.LSP


  This program draws welding symbol. This program supports only civil engineering style.
You must first select position of base line. You can draw more than one line.  
  


 
  Example 
Pick first base line point: {p1}    [Pick first point of base line, in this point lead line start]
Pick second base line point: {p2}       [Pick side of base line, length of base line is 12mm]
{Choose weld type on Dialog Box}
Pick first point: {p3}                  [First point of first lead]
Pick second point:                    [Finish first lead]
Pick first point: {p4}          [Pick first point of second lead]
Pick second point: {p5}                 [Pick end point of second lead]
