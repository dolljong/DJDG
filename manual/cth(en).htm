Change Text Height.

 
  Program name : CTH.LSP


  Ssometimes We need to change heitght of texts that are down on a drawing. 
This program can be used that moment. You can select texts that has certain height. 
You can select all texts. you can change to height that you want, 
or refer to height of exist text. 



 
 Example
Height/All/<Select>:           [type Enter key to select by mouse]
                            A          [use A option to select all texts on the drawing]
                            H          [use H option to select text that has certain height ]
Select text:
Select objects: {p1}                    [Select texts by using cross box]
Other corner: {p2} 2 found
Select objects:  2 found               [Finish selecting]
Pick text/<New height>: 500    [Set the new height as 500]
                            P          [use p option to refer to height of existing text]
