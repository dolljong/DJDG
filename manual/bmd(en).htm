Draw bending moment diagram
 
  File name : bmd.lsp


  This program draws the moment diagram using the input x coordinate and moment value. 
This program is made by using CAMBER.LSP program, so the usage method is is similar to CAMBER.LSP.
Please refer to CAMBER.LSP for detailed usage. You can draw the moment value of each load case them 
in combination or you can draw them individually. 



 
   Example 
Command: bmd
LEFT MARGIN (mm)<60>: 
l41-NODE / 2-CAMBER(S) / 1-COMBINATION(S) / 3-DRAWING(S) FOUND
Column width (mm)<20.0>:
