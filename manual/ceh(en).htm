Copy Entities to Hatch layer
 
  File name : CEH.LSP


  If you want to hatch only certain region in a drawing where complex lines are.
You have to copy it to a place, hatch it, and then move it back. 
This program makes the hatch work easier by making it invisible and only the hatch layer visible.
To finish the hatch operation and return to the original state, enter CEH1 in the command: state.
