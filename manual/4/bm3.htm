아래 그림과 같은 철근마킹을 해줍니다.

 
  Program name : BM3.LSP


  점철근이나 tie bar(평면도에서)를 마킹하는 것은 조금 짜증날 때가 있습니다. 왜냐하면 각각의 
점철근(또는 tie bar) center부터 마킹원까지 일일이 선들을 그려줘야하기 때문입니다. 이 프로
그램은 그러한 불편함을 없애기 위해 만들어졌습니다. 
    ◆ 주의사항
       ▷ 프로그램이 인식할 수 있는 entity는 donut(사실은 넓이를 가진 polyline임)과, 원 ,직선(tie 
          bar)입니다. 이 세가지 이외의 선택된 entity는 무시됩니다.

 
     사용예 
Select objects: {p1}~{p2} 5 found       [window opetion으로 점철근을 선택합니다]
Select objects:                        [선택을 끝냄]
Pick marking point: {p3}                [마킹원의 위치]
Enter Marking: bm3                     [철근이름, 소문자는 대문자로 변환됨]
Enter Rebar Dia: d29           [철근지름]
