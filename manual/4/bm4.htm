  Program name : BM4.LSP


아래 그림과 같은 형태로 점철근을 마킹해줍니다. 
    ◆ 주의사항
       ▷ 프로그램이 인식할 수 있는 entity는 donut(사실은 넓이를 가진 polyline임)과, 원 ,직선(tie 
          bar)입니다. 이 세가지 이외의 선택된 entity는 무시됩니다.

 
     사용예 
Select objects: {p1}~{p2} 5 found               [window option으로 점철근 선택]
Select objects:                                [선택을 끝냄]
Pick base line first point: {p3}                [보조선의 시작]
Pick base line second point: {p4}               [보조선의 끝]
Enter Marking: bm4                             [철근이름, 소문자는 대문자로 변환됨]
Enter Rebar Dia: d29                   [철근지름]
