Draw rebar that resists splitting force(plan view)
 
  File name : SB.LSP


  Split force occurs under the bearing on the abutment or pier. 
The rebar that is layed under bearing is called as split rebar.
Usually Scale of split rebar drawing is different from other scale of rebar drawing. 
It is picky to draw split rebar because you must increase the size. 
If you use this program. You don't need to consider that. 

 
   
   

  
  Example 
  command: Select SB function at [Rebar work] Menu
  You can Enter data by using dialog box. There is no something special.
