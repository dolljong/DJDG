Rebar marking type-1(Only circle)
 
  File name : BM1.LSP


  This program draws most simple bar mark like below image. 
Diameter of mark circle is 7mm(when it is printed. 
Name of rebar and diameter text is converted to Uppercase.


     
Example 
  Pick start point: {p1}        [Pick start point]
  Pcik end point: {p2}  [Pick position of mark. 
                                 this point is not center. It is bottom of mark circle]
  Enter marking: bm1   [Enter name of rebar. If you enter lowercase, program converts it to uppercase.]
  Enter rebar dia: d29 [Enter dia of rebar]
