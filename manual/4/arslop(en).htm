Array linear referencing inclined line.

 
  File name : ARSLOP.LSP


When you array on inclined line. If you array horizontaly. 
you must calculate inclined distance to array with AutoCAD array command.
It is kinda picky. This program arrays entity horizontaly or verticaly on the inclined line

 
     Example               
Select objects: {p1}1 found     [Select a entity to be arrayed]
Select objects:                        [finish selecting]
Pick start point: {p2}          [Pick start point]
Pick end point: {p3}                    [Pick end point]
Array distance: 600                    [Enter array distance]
X-direction/Y-direction: x     [Choose Horizontal or Vertical]
Select line/2P: {p4}                    [Select a line, 
                                  You can choose selecting inclined line or pick two points
10@600.0=6000.0                 [Program shows array distance, and number, total distance]
