Write numbers in sequence.
 
  Program name : BCN.LSP


  
This program number in sequence. If you draw the modeling in CAD for the structural report, 
you can draw the node number or element number using this function. 

 
 Example
Box/Circle/<None>: b                           [Choose shape. in this case box, You can choose circle or None]
Pick text/Text height<13.2307>: 2.5            [Enter height of text]
                                                Enter 'P' option to refer to height of existing text
Number1, ChangeNum/<Pick point>: {Pick a point}
Number2, ChangeNum/<Pick point>: {Pick a point}
Number3, ChangeNum/<Pick point>: {Pick a point}
Number4, ChangeNum/<Pick point>: c             [Enter 'C' option to change number]
Enter New Number: 5                            [Enter new number, in this case we will restart with 5]
Number5, ChangeNum/<Pick point>:         {Pick a point}
Number6, ChangeNum/<Pick point>:         {Pick a point}
Number7, ChangeNum/<Pick point>:         {Pick a point}
Number8, ChangeNum/<Pick point>:               [Type Enter key to finish command]
