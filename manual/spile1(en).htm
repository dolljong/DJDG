Draw steel pile (section)

 
  Program file name : SPILE1.LSP

 
   Example
  command: spile1
  Pick start point: {p1}         [Pick start point of pile. Under line of footing]
  Pile diameter (mm): 406.4   [Pile diameter (mm)]
  Pile thickness (mm): 9      [Pile thickness (mm)]
  Pile length (m): 5          [Pile length (m)]
  Angle/<1:?>: 5        Angle of pile
                                          [If you enter number then draw pile angle of '1: Number'
                                          "A" : Draw angle(degree) of pile 
                                          "0" : Draw vertical pile]
