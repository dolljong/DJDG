사칙연산을 해주고 그 결과를 도면에 기입해줍니다.
 
  Program name : CALT.LSP


  때로는 도면내에 있는 숫자들의 합이나 곱의 결과를 알아야할 때가 있습니다. 그럴 땐 보통 계
산기를 들고서 일일이 계산을 하는데 여간 불편한 것이 아닙니다. 이 프로그램은 도면내에 있는 
숫자들을 선택해서 그 합, 차, 곱, 나누기들을 해줍니다. 그리고 그 결과를 도면에다 기록할 수도 
있습니다.  연산은 누적됩니다. 즉 앞의 연산결과를 계속 보여주면서 다음 연산을 대기합니다. 아
래의 사용법을 보시면 이해가 되시리라 생각됩니다. 설명하기 힘들 때는 이런식으로 슬쩍 넘어가
야지..

 
 사용예-1
Summation<Select text> :                [423.500선택]
Total = 423.500
Sum / + / - / / /Insert/Exit: *                [곱하기 기능 선택]
Total = 3324.475                                [현재의 계산 결과가 화면에 표시됨]
Sum / + / - / / /Insert/Exit: i                [도면에 현재의 연산 결과 쓰기 선택]
Number of digits of right of decimal point (0~8): <3>: 3 [소수점 3자리]
Pick insert point:                      [연산 결과의 insert point 찍어줌] (오른정렬) 



 사용예-2
Summation/<select text>: s     [여러개의 숫자 더하기 기능 선택]
Select text:                    [화면에서 3324.475, 1986.050, 981.250, 745.750, 404.275선택]
Select object:                         [선택을 끝내기 위해 Enter입력]
5 Found
Summation: 7441.800                     [화면에 현재 계산 결과가 표시됨]
Sum / + / - / / /Insert/Exit: i        [도면에 현재의 연산 결과 쓰기 선택]
Number of digits of right of decimal point (0~8): <3>: 3?       [소수점 3자리]

Pick insert point:                      [연산 결과의 insert point 찍어줌] (오른정렬)
