TABLE을 그려줍니다.
 
  Program name : TABLE.LSP


  도면작업을 하다보면 table을 그려야하는 경우가 종종 있습니다. 이 명령은 주어진 칸수와 줄수
를 가진 table을 만들어줍니다. 
·줄간격은 7mm로 고정시켰습니다.

 
 사용예
  Insert point:         [테이블의 왼쪽 상단이 될 점을 찍어줍니다.]
  Column number: 5     [칸 수를 입력합니다. 여기서는 5칸]
  Row number: 6        [줄 수를 입력합니다. 여기서는 6칸]
                                   Pitch of Column(mm)<20>: 15? [칸간격을 입력합니다. default
                                   는 20mm, 여기서는 15mm]
