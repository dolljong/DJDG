철근재료표를 만들어줍니다.
 
  Program name : HBLIST.LSP


  사실 철근재료표를 그리는 방법은 매우 단순합니다. 철근의 길이들과 갯수들을 곱해서 철근의 
총중량을 구하는 것이죠. 국민학생들도 할 수 있는 작업이지요. 그래서인지 계산기 두드리기를 귀
찮아하는 사람들이 있습니다. 바로 그런 사람들을 위해서 이 프로그램은 만들어졌습니다. 철근의 
길이와 직경만 써주면 재료표를 완성시켜줍니다. 이 프로그램을 사용하여 재료표를 만들면 처음 
만들때도 편하지만 특히 이미 만들어진 재료표를 수정할 때 상당히 편합니다. 한개의 철근 길이
때문에 모두 다시 계산하는 대신에 데이타 한줄만 바꿔서 프로그램을 다시 돌리면 되니까요. 
  
 
  ◆ 어떤 기능을 가지고 있는가?
      ▷data file의 data들은 ,로 구분됩니다. 공백은 무시되므로 보기좋게 편집하시면 됩니다.
      ▷보통의 철근재료표에는 소문자를 쓰지 않으므로 소문자로 입력된 data는 모두 대문자로 바
        꿔줍니다. Caps Lock키가 눌러져 있는지 신경을 안써도 되겠죠?
      ▷철근직경을 기준으로 정렬하고 다시 철근번호를 기준으로 정렬되므로 입력순서에 신경쓸 필
        요가 없습니다. 아무생각없이(?) Bar detail도에 나온 순서대로 입력하면 OK!
      ▷철근길이의 단위는 m단위입니다. 길이가 1m이하일 경우 재료표에 0.500이 아닌 500으로 표
        시해줍니다. 물론 중량계산에서는 정상으로 계산되니 걱정하지 마시고...
      ▷재료표만으로 한장의 도면을 만드는 경우도 있지만, 때에 따라서는 배근도 옆에다 재료표를 
        그리는 경우도 있습니다. 그때를 위해서 scale factor를 줄 수 있도록 했습니다.
      ▷재료표 한줄의 높이는 7mm이고 글씨의 크기는 2.5mm입니다. (물론 출력했을 때 기준으로..) 
     
  ◆ 주의사항
      ▷AutoCAD에서 한글을 구현하기 힘들었기 때문에 부득이 블럭을 이용하였습니다. 즉 철근재
        료표에는 네개의 블럭이 사용됩니다. 대가리(Header,제목과 테이블의 첫줄)와 소계, 총계입니
        다. 블럭의 이름은 각각 \BLOCKS\BHEAD.DWG, \BLOCKS\BGRDT.DWG, \BLOCKS\BSUBT.DWG
        입니다.

사용예-2  도면내의 text enetity를 이용하는 경우
File/Entity <File>: E          [도면내의 text를 통해서 철근 data입력받습니다]
select entity: {p} {p2}        [cross box를 이용하여 도면내의 text선택]
select entity:               [입력을 마치기 위해서 엔터입력]
insert point:                   [insert point 찍어줌 (table의 왼쪽상단)]
scale factor: 50             [알맞은 scale factor]

 


 
   ※ data를 입력하는 방법은 위와 같이 두가지 입니다. 파일을 이용하거나 도면내에다 text
   명령 또는 dtext명령을 이용해서 입력하는 방법입니다. 파일을 이용하는 방법의 장점은 편집이 
   간편하다는 것이고, 도면내에다 직접 입력하는 방법의 장점은 도면내에 data가 있으므로 데이
   타파일을 따로 챙길 필요가 없다는 것입니다. 사용자의 취향에 맞추어 쓰시면 되겠습니다.
