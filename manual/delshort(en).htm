Delete all lines that is shorter then specified length  
 
  File name : delshort.lsp


   Delete lines below a certain length. When drawing, there are cases where small entities are broken(?), This program 
cleans up these entities.

 
   Example
Command: delshort
All/<Select>:     [Select entities, All = All entities | Enter = select entities]
Select Lines: {Pick p1} {pick p2}   [Select entities by using Window box]
Select objects:  Specify opposite corner: 16 found  [Finsh selecting]
Select objects:  16 Line found        [Program shows how many entities are selected, in this case 16 entities were selected]
Enter Length: 300                     [Enter length, in this case 300 was entered, program deletes entities that is shoter then 300]
8 Line Deleted
