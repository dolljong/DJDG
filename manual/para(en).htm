Draw Parabola.
 
  File name : para.lsp


  This program draw parabola by using dialog box 



   Example 
Command: para
Pick End point: {p1}  [PIck end point of parabola]
Pick Mid point: {p2}  [Pick mid point(vertex of parabola]
{choose options on the Dialog Box}
  
Property of Parabola: display formular of parabola.
Divid Num: division number of parabola. 
Both: Draw both side of parabola
Cap: Draw perpendicular line on the end of parabola.
Length: Length of Cap line
