아래 그림과 같이 철근마킹을 해줍니다.
 
  Program name : BM1.LSP


  아래의 그림과 같이 가장 단순한 철근의 마킹을 해줍니다. 마킹원의 크기는 지름 7mm(출력하
였을 때 기준) 입니다. 철근명과 철근 DIA는 소문자로 입력하여도 대문자로 고쳐서 만들어집니다.


     
사용예 
  Pick start point: {p1}        [마킹할 철근을 찍어줍니다]
  Pcik end point: {p2}  [마킹원이 위치할 점을 찍어줍니다. 
                                 마킹원의 중심이 아니라 하단입니다.]
  Enter marking: bm1   [철근이름을 입력합니다. 소문자는  대문자로 변환됩니다.]
  Enter rebar dia: d29 [철근의 직경을 입력합니다]
