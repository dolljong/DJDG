Solid hatch
 
    File name : SHATCH.LSP


  Using the hatch command can sometimes be inconvenient. For example, as shown on the left below 
to hatch the header of the table, We have to do hatch command tree times on each column. or We have to draw 
rectangle with four points. and do hatch command. With this program you only need to Pick four points. If you 
pick the outer points of the part you want to hatch, the hatch will be done 
 


 Example 
Command: shatch
Select objects: {Pick p1} {Pick p2} {Pick p3} {Pick p4}  [Pick points in order]
Select objects:           [Type Enter key to finish Picking]
