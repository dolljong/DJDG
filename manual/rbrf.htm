기둥 주철근의 정면도를 그려줍니다.
 
  Program name : RBRF.LSP


  교각기둥의 주철근은 원형으로 배열되있으므로 정면도에서 그것을 표현해주기가 쉽지 않습니
다. 단면도를 그리고나서 각 점철근에서 라인을 그려 사용하곤 했습니다. 이 프로그램은 그러한 
수고를 덜어줍니다. 

 
  사용예 
    Select first line: {p1}                     [첫번째 철근선 선택]
    Select second line: {p2}                    [두번째 철근선 선택]
    All round/Half round <All round>:      [기둥전체를 보이기 선택, 트랙형 기둥의 원형
                                                부같이  반쪽을 보이기 위해서는 "H" 입력]
    Number of Rebar: 70                        [철근의 갯수 입력]
