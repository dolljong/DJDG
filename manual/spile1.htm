강관파일(단면) 그리기(1)

 
  Program file name : SPILE1.LSP

 
   사용예
  command: spile1
  Pick start point: {p1}         [파일의 시작점 찍어줌, footing 하단]
  Pile diameter (mm): 406.4   [강관파일지름(mm)]
  Pile thickness (mm): 9      [강관파일 두께(mm)]
  Pile length (m): 5          [강관파일의 길이(m)]
  Angle/<1:?>: 5               [파일의 경사입력
                                          숫자를 입력할 경우 '1: 몇'의 pile 그림
                                          "A"를 입력하면 각도(degree)를 직접 입력 
                                          "0"을 입력하면 직항으로 그림]
