일정길이 이하의 라인 지우기  
 
  파일명 : delshort.lsp


  일정길이 이하의 line들을 지워줍니다. 도면을 그리다보면 작은 엔티니들이 널부러져(?)있는 경
우가 있는데 이 엔티티들을 말끔히 지워줍니다.

 
   사용예
Command: delshort
All/<Select>:     [적용할 엔티티를 선택, All=전체엔티티 , Enter=엔티티선택]
Select Lines: {Pick p1} {pick p2}   [두점을 찍어 window box로 엔티티 선택]
Select objects:  Specify opposite corner: 16 found  [선택끝냄]
Select objects:  16 Line found        [16개의 엔티티가 선택됨]
Enter Length: 300                     [300이하의 line들을 지우기 위해 300을 입력한다]
