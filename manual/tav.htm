  Program name : TAV.LSP


  토목도면에는 유난히 테이블이 많은 것 같습니다. 테이블에 숫자를 모두 써넣고 보면 어떤 땐 
숫자들이 삐뚤빼뚤하게 배열될 때가 있습니다. 그것들을 가지런히 정렬하기도 쉽지않죠? 이 프로
그램은 이런 숫자들을 왼쪽, 가운데, 오른쪽을 기준으로 일렬로 가지런히 정리해주는 기능을 가지
고 있습니다.

 
 사용예
Select objects: {p1}                    [cross box를 이용하여 text선택]
Other corner: {p2} 3 found      
Select objects:                        [선택을 끝내기 위해 ?]
<Left>/Middle/Right: m         [middle point를 기준으로 정렬]
Pick Point: {p3}                        [middel point를 찍어줌]
