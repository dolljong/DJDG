Solid hatch를 간편하게
 
    파일명 : shatch.lsp


  hatch 명령을 사용하다보면 가끔 불편할 때가 있습니다. 예를 들어 아래의 왼쪽 그림과 같이 
여러개로 칸이 나누어진 테이블의 header부분을 hatch하려면 hatch-pick point로 hatch를 세 번하
든가 아니면 polyline이나 rectangle명령으로 hatch하고 싶은 영역을 사각형을 그린 다음에 
hatch-entity로 hatch해야합니다. 이 프로그램을 사용하면 네점만 찍어주면 됩니다. hatch하고싶은 
부분의 외곽점을 차례로 찍어주면 hatch해줍니다.  


  
 사용예 
Command: shatch
Select objects: {Pick p1} {Pick p2} {Pick p3} {Pick p4}  [차례대로 점을 찍음]
Select objects:           [선택을 끝내기 위해서 Enter입력]
