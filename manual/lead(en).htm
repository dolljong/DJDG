Draw leader
 
    File name : LEAD.LSP


  Draw leader. Length of leader underline is determinded as thel length of text that user enters.

 
   Example
  First point: {p1}     [Pick first point. end point of Arraw head]
  Second point: {p2}    [Pick second point. Start point of under line]
  LEADER               [Enter text through DTEXT command]
               When text input is finished, 
                   Program determines lenght of underline by referencing the text length.
