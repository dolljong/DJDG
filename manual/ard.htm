구간 Array를 해줍니다.
 
  Program name : ARD.LSP

AutoCAD의 Array명령은 x방향갯수, y방향갯수, x방향간격, y방향간격 등 입력사항이 많기 때문에 
배든도의 점철근같이 한방향으로 array하는 경우에는 불편할 수 있습니다. 이 프로그램은 구간과 
간격을 정해주면 개수를 구해 array해주기 때문에 미리 계산기를 두드려 개수를 구할 필요가 없습
니다. 단 x방향과 y방향만 가능합니다.

 
     사용예               
Select objects: {p1} 1 found           [array할 엔티티 선택]
Select objects:                        [선택종료]
Pick start point: _cen of {p2}        [시작점 입력]
Pick end point: {p3}                    [끝점입력] 프로그램이 x,y길이를 측정해서 긴쪽방
                                                      향으로 array방향을 정함
Distance = 4575.84                     [측정한 거리를 사용자에게 보여줌
Array distance: 600                    [array 간격 입력]
Remainder = 375.836                    [array를 실행하고 남은 거리를 표시해줌]
