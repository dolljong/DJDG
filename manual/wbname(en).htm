Do Wblock saving file name is selected text...
  File name : wbname.lsp

  Normally, when working with drawings, you do not draw one drawing in one file, 
  but several drawings in one file. However, when you deliver CDs, you should not put several drawings in one file like this.
  Because it's hard to search. So We work multiple drawing when we work, but when We deliver later, We have to cut it separately. 
  Usually with the WBLOCK command. But it is inconvenient to enter filenames and there is a possibility of typos.
  this program can name a file using the text in the drawing, You can select file name by drawing number or drawing name. 


  
 Example 
Command: wbname
Select objects: {Pick s1} 1 found  [Select text that will be file name]
Select objects:  
                                       [Type Enter key to finish selecting, 
						 If you several text, file name is made by appended that texts]
Insertion base point: {Pick p1}  [Pick insertion point]
Select objects: {Pick p1} Specify opposite corner: {Pick P2} 5 found 
                                     [Select entity that you want to make block by using window box]
