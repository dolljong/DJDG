LINE의 길이를 도면에 기입해줍니다.
 
  Program name : WLEN.LSP


  2PD.LSP과 마찬가지로 모델링 작업등에서 line의 길이를 구해서 도면에 기입해줍니다. SCALE 
FACTOR를 이용하여 구하여진 Line의 길이에 일정한 수를 곱해서 기입할 수 있습니다. 

 
 사용예
Text height : 150              [text의 크기]
Text angle : 90                [text의 회전 각도]
Scale factor: 0.001            [scale factor]
Select Line: 
Select object: {p1}             [첫번째 line 선택]
Pick insert point {p2}  [line길이를 기입할 위치, text의 mid point]
Select object: {p3}             [두번째 line 선택]
Pick insert point {p4}  [line길이를 기입할 위치, text의 mid point]
Select object: {p5}             [세번째 line 선택]
Pick insert point {p6}  [line길이를 기입할 위치, text의 mid point]
Select object:                 [명령을 끝내기 위해 Enter]
