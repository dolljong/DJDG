Do Measure on Line.

 
  Program name : MEASL.LSP


  It is similar to MEASA.LSP. But it is for line not arc. 
You can choose point or block as a pointer of measure. 
  


 
 Example-1
Pick start point: end of {p1}           [Pick start point]
Pick end point: nea to {p2}             [Pick end point]
<Segment length>/Block: 500    [Enter distance of MEASURE]
Total length= 2458.16           [Program shows distance between two points]
Number of segment= 4                    [measure number is Shown]

 
 
 

 

 
 Example-2
Pick start point: end of {p1}           [Pick start points]
Pick end point: int of {p2}             [Pick end points]
<Segment length/Block: b               [Enter 'b' to choose block option]
Block name: star                       [Enter name of block]
Align block with object? <Y>: y        
            [Enter option of align. If you choose align blocks is aligned to angle of two points]
Segment length: 500                    [Enter measure distance]
Total length= 2323.77           [Program shows distance between two points]
Number of segment= 4                    [Number measured is shown]
