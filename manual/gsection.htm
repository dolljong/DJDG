Gsection data 파일 만들기
 
  파일명 : gsection.lsp


  Gsection(General section)은  (주)DM엔지니어링에서 제작하여 공개한 토목설계용 전용 프로
그램 중 하나로서, 임의 형상의 단면에 대한 단면물성치(단면적, 단면2차모멘트 등)을 구해주는 
프로그램입니다. 이 프로그램의 data파일(*.sec)은 구하고자하는 단면을 이루는 단일폐곡선의  각 
node x,y 좌표값들을 나열해주는 형태인데 단면이 복잡한 경우 x,y좌표를 일일이 구하는 것도 간
단한 일이 아닙니다. [다정다감]의 gsection.lsp는 캐드상에서 작성된 그림을 이용하여 간단히 임
의 단면의 좌표값들을 구하여 gsection.exe용 data파일로 만들어주는 기능을 가지고 있습니다. 


  
 사용예 
  아래 그림과 같은 P.S.C BOX 단면의 section property를 구해보도록 하겠습니다.  
  




 
  작업에 들어가기 전에 먼저 알아두어야 할 것은 gsection.exe는 단일 폐곡선만을 인식한다는 
것입니다. 위의 그림과 같이 중공이 있는 단면은 단일 폐고선이 아니므로 단일 폐곡선으로 만들 
수 있도록 아래 그림과 같이 3개의 선을 추가해줍니다. 이제 어느 한점에서 시작해서 선을 따라
가면 다시 시작했던 점으로 돌아 올 것입니다. 이렇게 단일 폐곡으로 만들 수 있는 준비를 해줍
니다. 물론 중공단면이 아닌 경우엔 이런 작업이 필요없겠죠

 
  
 
 
  이제 [다정다감] gsection을 실행시킵니다.
Command: gsection
Pick point or [Select polyline]: {Pick p1}  [폐곡선 내의 한점을 찍어줍니다]
                       S{Select s1} [이미 만들어져있는 폐합 polyline을 선택합니다.
40 node found                           [40개의 node가 발견되었다는 메시지 나옴]
Pick Original point: {Pick p2}       [기준점 찍어줌]
Enter Title: P.S.C BOX                [gsection.exe에서의 제목 입력]
Enter scale factor: 0.001            [gsection.exe로 넘어갈 때 scale] 



 
 
그러면 아래 그림과 같이 저장할 파일이름을 물어보는 box가 뜹니다. 적당한 이름을 정하여 저장
합니다. 


 
 

 
 
gsection.exe를 실행시키면 아래아 같이 결은 결과를 얻을 수 있습니다.

 
 
 
 
Bpoly명령사용하기
 
bpoly명령은 폐합 polyline을 만들어주는 명령입니다. 프로그램이 "Polyline must be 
closed"란 에러메세지와 함께 끝난다면 bpoly를 이용하여 폐합 polyline을 만든 다음에 
gsection에서 Select polyline옵션을 사용하여 영역을 정해줄 수 있습니다. 

command: bpoly[Enter]
