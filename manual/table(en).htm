Draw table.
 
  Program name : TABLE.LSP


   This program draws table with given column numbers and row numbers. Height of row is 7mm.

 
 Example
  Insert point:         [Pick upper left corner of table]
  Column number: 5     [Enter number of column. in this case, 5 columns]
  Row number: 6        [Enter number of rows. in this case 6 rows]
  Pitch of Column(mm)<20>: 15 [Enter width of column. default is 20mm, in this case, 15mm]
