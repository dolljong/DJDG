베이스 콘크리트를 그려줍니다.
 
  파일명 : basec.lsp


  지반위에 설치되는 기초의 아랫면을 고르게 만들기 위해 타설되는 베이스 콘크리트는 보통 
100mm두께로 타설합니다. 버림콘크리트라고 하기도 합니다. 베이스 콘크리트의 두께를 입력하도
록 하였고 구조물의 양쪽으로 여유있게 타설하는 폭은 100mm로 고정시켰습니다.
 
   사용예-1 (기초의 저면선을 이용하여 그리기)
Command: basec
Select Line/<2p>: {select s1}
Thickness <100>: 
 
   사용예-2 (기초저면 양끝 2점을 이용하여 그리기)
Command: basec
Select Line/<2p>: 2
Pick first point: {pick p1}
Pick second point: {pick p2}
Thickness <100>:
