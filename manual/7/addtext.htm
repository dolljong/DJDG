숫자 text에 일정 값 더하기
 
  파일명 : addtext.lsp


  이미 기입된 숫자 text을 원하는 값만큼 증가시키거나 감소시키는 기능입니다. 하부 일반도 작
업이 끝난 상태에서 종단이 변경되었다면 모든 교좌면 Elevation이 일정한 값만큼 감소하거나 증
가해야 할 것입니다.  이런 경우에 사용할 수 있습니다.

 
   사용예
  Command: addtext
  All/<Select>:           [All: 모든text 선택,  원하는 text만 선택]
  Select text: w{p1} {p2} [Window box를 이용하여 Text선택]
  Select objects:         [선택종료]
  Enter Add value: 0.2   [증감값 입력]
