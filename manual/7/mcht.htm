text를 순서대로 수정해줍니다.
 
  Program name : MCHT.LSP


  Table에서와 같이 text가 많은 도면에서 여러 text를 수정하기 위해서 text를 하나 하나 선택해서 
수정하는 것은 여간 짜증나는 작업이 아닐 수 없습니다. 이 프로그램은 여러개의 text를 선택한 
후 좌에서 우로 또는 위에서 아래로 순서대로 수정할 수 있도록 해줍니다.

 
 사용예
Horizontal/Vertical <V>:               [위에서 아래로 수정]
Select objects: {p1}                    [cross box를 이용하여 text선택]
Other corner: {p2} 4 found              
2 were filtered out.                    [text가 아닌 entity는 걸러내짐]
Select objects:  2 text found  [text선택을 끝냄]
Enter text <TEXT1>: text1-1           [화면의 'text1'이 반전됩니다. 이때 새로운 text내용을 
                                        입력]
Enter text <TEXT2>: text2-1           [두번째 text내용을 입력]
