Text내용 복사하기
 
  파일명 : ctext.lsp


도면작성을 하다보면 종종 같은 내용의 text를 여기저기 써야할 경우가 많습니다. 이때 일일이 
바꿔주지 않고 한 개만 바꾼다음 나머지는 똑같이 복사하는 기능입니다. 
  


 사용예
Command: ctxt
Pick Source Text: {select p1}  [원본 text를 선택]
Select destination text: {select s2}, {select s3} [복사될 text들 선택]
Select objects:   [선택종료]
2 Text Found
2 Text Modified  [몇개의 text가 변경되었다고 알려줌]
