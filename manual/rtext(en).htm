Write text on circle or arc
 
    File name : SLOPL.LSP


  This program control the rounded text as a single entity. 
The rounded text is a block. But user can't insert this block. Because this block is an unnamed block. 
AutoCAD has some unnamed blocks. Hatching and Dimension are that. 
This program ask center of circle (p1) and center of text(p2). and text height.
 
  Example
  Pick center point: {p1}       [Pick center of circle]
  Pick insert point: {p2}       [Pick insert point]
                          Program decides the angle of text by referencing to center of circle and insert point.
  Text height: 2.5            [Enter text height]
  TEXT: This is a round text       [Enter text]
