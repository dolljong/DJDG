두께를 가진 강재가 겹치는 부분을 정리해줍니다.
 
  Program name : TRIMB2.LSP


  위의 TRIMB와 trim하는 기능은 같으나 TRIMB2에서는 잘라진 직선의 끝을 마무리하여줍니다. 즉 
잘라지는 선은 잘라지지 않는 선에서 일정 거리만큼 떨어진 곳에서 잘라지고, 그 잘라진 끝을 이
어줍니다. - 이하 독백 -  음.. 글로 표현하려니까 정말 어렵군... 이런땐 항상 생각나는 말이 있
지..  몇년전 작고한 신동우화백이 어느 필기구 광고에서 한 말인데... 대충 이런 뜻이었지.."제가 
그림을 좋아하는 이유는 천마디의 말로 표현해야하는 것을 한장의 그림으로 나타낼 수 있기 때문
이지요" - 이상 독백 -   아래 그림을 보시지요

 
 사용예
select first entity: {p1}               [잘라지지 않는 plate (예:steel box 수직보강재)]
select first entity: {p2}       
Ofset distance<0.0>: 30                [30mm만큼 이격시킴]
Pick first point: {p1}          [잘라지는 plate (예:수평보강재)]
Pick second point: {p2}         [{p1}과 {p2}를 연결하는 선과 만나는 선을 정리해줌]
Pick first point: {p3}          [계속해서 두점을 이용하여 정리함]
Pick second point: {p4}
Pick second point:             [명령을 끝내기 위해서 Enter입력]
