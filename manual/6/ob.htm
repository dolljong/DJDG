치수선을 기준선에 맞춰 Oblique하기
 
  파일명 : ob.lsp


지반위에 설치되는 기초의 아랫면을 고르게 만들기 위해 타설되는 베이스 콘크리트는 보통 
100mm두께로 타설합니다. 버림콘크리트라고 하기도 합니다. 베이스 콘크리트의 두께를 입력하도
록 하였고 구조물의 양쪽으로 여유있게 타설하는 폭은 100mm로 고정시켰습니다.
 
   사용예-1 (기존에 그려전 선을 이용하는 경우)
Command: ob
Select Dimension: {select s1}   [oblique할 치수선을 선택합니다]
Select objects: 1 found
Select objects:          [치수선 선택을 끝냅니다]
Select line/2p: {select s2}     [기준이 되는 선을 선택합니다]
 
   사용예-1 (기존에 그려전 선을 이용하는 경우)
Command: ob
Select Dimension: {select s1}   [oblique할 치수선을 선택합니다]
Select objects: 1 found
Select objects:          [치수선 선택을 끝냅니다]
Select line/2p: 2             [두점을 이용하는 옵션을 선택]
Pick start point: {Pick p1}     [첫째점 선택]
Pick end point: {Pick p2}       [둘째점 선택} 




 
※ 기준이 되는 두점을 찍을 때 p2, p1순으로 찍으면 치수선이 거꾸로 나올까요?
아닙 니다. p1,p2순으로 찍으나, p2,p1순으로 찍으나 결과는 같습니다.
