DIMENSION을 확대, 축소해줍니다.
  
 
    Program name : CDS.LSP


AutoCAD를 이용해서 1:100으로 그려진 도면을 1:50으로 바꾸려면 어떻게 해야 할까요? CAD에서 
Scale만 나오면 헷갈려 하시는 분들이 많더군요. 구조물을 mm단위로 그렸다면 치수선 크기를 반
으로 줄이면 되겠지요? 그렇다면 치수선의 크기를 반으로 줄이는 방법은? 치수선을 그릴 때 dim
명령을 이용했다면 의외로 간단합니다. command: dimscale? 명령으로 dimscale값을 50으로 바꾸
어준 다음에 dim: update?명령으로 갱신을 하면 됩니다. 하지만 아래의 두번째 그림과 같이 치수
선의 위치가 문제입니다. 이때 이 프로그램을 사용하여 치수선의 위치를 아래의 세 번째 그림과 
같이 옮겨주면 1:50도면이 되는 것입니다. 여러 가지 면에서 line과 donut, text 명령을 이용해서 
치수선을 그리는것보다 DIM명령을 이용하는 것이 유리하다고 생각합니다.

 
   사용예
Select objects: {p1}1 found             [SCALE을 조정할 치수선 선택]
Select objects:                        [선택을 끝냄]
Enter Scale: 0.5                       [SCALE값을 입력함. 50/100=0.5]
