Draw lane mark
 
  File name : lanemark.lsp


  It is not easy to draw lane mark on inclined line.
This program draw lane mark on inclined line.  
  

 
   Example
  command: lanemark
  Select line: {s1}                     [Select a line of road surface]
  Pick X-position: {p1}                         [Pick a point. Program refers to x coordinate of that point.]
  Pick side: {p2}                               [Pick direction] 
  Enter Number: 2                        [Enter number of lanes]
  Enter Width of lane: 3600      [Enter width of lane]
