선의 일부를 숨은 선으로 고쳐줍니다.
 
  Program name : BDASH.LSP


  구조물을 그리다보면 구조물이 위/아래로 교차하는 경우가 있습니다. 그럴땐 아래쪽 구조물 선
은 보이지 않으므로 점선으로 나타내야 합니다. 이 프로그램을 이용하면 선을 잘라내고 다시 
hidden type의 선으로 두 점을 이어주어야 하는 번거로움 없이 선과 두점을 선택해주기만 하면 됩
니다.

 
 사용예
Select Line: {p1}               [line 선택]
First point: int of {p2}        [첫점 찍어줌]
Second point: int of {p3}       [두째점 찍어줌]
command:                       [명령반복]
Select Line: {p4}               [line 선택]
First point: int of {p5}        [첫점 찍어줌]
Second point: int of {p6}       [두째점 찍어줌]
