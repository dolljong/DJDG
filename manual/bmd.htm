모멘트선도(BMD) 그리기
 
  파일명 : bmd.lsp


  입력된 x좌표와 모멘트값을 이용하여 모멘트선도를 그려줍니다. 이 프로그램은 camber도 그리
는 프로그램을 약간조정해서 만든 것이기 때문에 사용법은 camber도 프로그램(camber.lsp)과 거
의 비슷합니다. 자세한 사용법은 camber.lsp를 참고하시기 바랍니다. camber와 다른 것은 모멘트
값을 그래프에다 표시해준다는 것입니다. 물론 camber와 마찬가지로 각 하중 case의 모멘트값을 
조합해서 그릴 수도 있고, 각각 그릴 수도 있습니다. 



 
   사용예 
Command: bmd
LEFT MARGIN (mm)<60>: 
l41-NODE / 2-CAMBER(S) / 1-COMBINATION(S) / 3-DRAWING(S) FOUND
Column width (mm)<20.0>:
