Insert Design Method(Allowable/Ultimate)
 
  program files name : ALL-ULT.LSP


This program insert two blocks below
  \BLOCKS\ALLOW.DWG : [Allowable]
  \BLOCKS\ULTIM.DWG : [Ultimate]
This program use font of GHS.SHX. If you change font then you can modify above files

 
   Example
  Allowable/Ultimate: A  [Select One Allowable or Ultimate]
  Pick upper right corner:      [pick upper right corner of border]
                                        This program reference AutoCAD Environment Variable 'DIMSCALE'
