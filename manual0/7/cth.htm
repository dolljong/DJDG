text크기를 바꾸어줍니다.

 
  Program name : CTH.LSP


  CAD로 작업을 하다보면 가끔 도면에 쓰여진 글씨들의 크기를 모두 또는 일부만 바꾸어줄 필요
가 있습니다. 그럴때 이 프로그램을 사용할 수 있습니다. text를 선택할 때 어느 일정한 크기를 가
진 text만 선택할 수 있고, 도면내의 모든 text를 선택할 수도 있습니다. 그리고 원하는 text의 크
기를 직접 입력할 수도 있고, 이미 쓰여진 text를 찍어 참조할 수도 있습니다. 



 
 사용예
Height/All/<Select>:           [마우스로 선택하기 위해 ?]
                            A          [도면내의 모든 text선택]
                            H          [어떤 크기의 text를 모두 선택]
Select text:
Select objects: {p1}                    [cross box를 이용하여 text선택]
Other corner: {p2} 2 found
Select objects:  2 found               [선택을 끝냄]
Pick text/<New height>: 500    [새 크기를 500으로]
                            P          [이미 쓰여진 text의 크기를 참조함]
