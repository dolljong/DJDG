수평치수선을 그려줍니다.

 
  Program name : DH.LSP


  도면 작업중 가장 많은 시간을 소요하는 작업이 무엇일까요? 음...  물론 사람들에 따라서 다르
겠지만, 저는 망설임 없이 치수선 작업이라고 대답하겠습니다. 사실 치수선이야말로 도면에서 가
장 중요한 것이라 하겠습니다. 도면은 설계자의 생각을 현장의 시공자에게 정확하게 전달하는 게 
목적이라고 할 수 있습니다. 현장의 시공자는 물론 그림도 보겠지만 치수에 의존해서 모든 작업
을 하므로 그림보다는 숫자가 중요하다고 하겠습니다.  
  


 
  ◆ 프로그램의 기능 및 주의사항
      ▷ 기존의 AutoCAD의 치수선 명령은 한개의 치수선만 그릴 수 있었습니다. 하지만 대부분의 
     도면엔 치수선이 연속해서 나옵니다. 연속치수기입이 쉽도록 프로그래밍 했습니다. 
      ▷ 두점을 찍거나 엔티티를 선택해서 치수선의 길이를 정해주는 기존방법이 불편할 때가 많아
     서 사용자가 길이를 직접 입력하여 치수선 길이를 정할  수 있도록 했습니다.
      ▷ 치수선이 구조물에서 떨어진 거리를 사용자가 직접 찍어주기보다는 1단, 2단... 등의 옵션으
     로 택할 수 있게했습니다.
      ▷ 골뱅이(@)기능을 만들어서 여러개의 반복되는 치수선을 하나로 묶어 표현할 수 있도록 했
     습니다. 골뱅이(?)기능을 쓰려면 dimension text를 물어올 때 @100과 같이 @나누는길이 형
     식으로 입력하면 프로그램이 나눠진 갯수를 계산하여 치수선을 그려줍니다. 이때 치수보조
     선 사이에 text가 들어가지 않으면 10@100부분과 =1000부분을 잘라서 위 아래에다 따로 기
     입해줍니다. 골뱅이(@)옵션을 사용할 때 주의 사항은 아래의 예와 같은 경우입니다. 골뱅이 
     옵션을 사용할 때 이미 찍어준 두점의 길이와 곱하기 연산결과가 틀릴 경우입니다. 이 때 
     프로그램은 틀린대로 도면에 그려줍니다. 일종의 경고 기능이라 하겠습니다. 예를 들어서 두
     점을 찍었는데 그 길이가 1050mm이고, 골뱅이(@100)옵션을 선택해서 나누기 길이 100을 
     주었다고 한다면, 프로그램은 1050/100의 결과인 10.5중 소수점 이하인 0.5를 버리고 10을 
     사용합니다. 그래서 결국 "10@100=1050"이란 결과를 내보냅니다.  
     


 
 사용예
Pick first point/Object: {p1}   [치수선의 시작점]
Pick Dimension line side: {p2}  [치수가 쓰여질 위치 위/아래 (여기서는 아래쪽)]
Level <1>:                             [치수선 단계 (여기서는 1단)]
Distance or RETURN to Pick point: 950  [알고있는 거리는 직접 입력함]
Dimension text <950>:          [만일 text를 고치고 싶으면 여기서 바꿀 수 있음]
Distance or RETURN to Pick point/eXit:         [거리를 모른다면 점을 찍어 입력하기 위해 Enter]
Pick point: mid
of {p3}                                 [한점을 찍어줌]
Dimension text <17.600>: @2200 [프로그램이 계산한 거리를 보여줌, 골뱅이 옵션 선택]
Distance or RETURN to Pick point/eXit: 950      [알고있는 거리는 직접 입력함]
Dimension text <950>:          [950을 그대로 사용함]
Distance or RETURN to Pick point/eXit: x       [DH명령을 끝냄]
{다시 DH명령을 시작함}                  [2단 치수선을 그리기 위함]
Pick first point/Object: {p1}   [시작점을 찍어줌, 이때 1단 치수선의 node를 찍어줌]
Pick Dimension line side: {p2}  [아래쪽/위쪽을 선택 (여기서는 아래쪽)]
Level <1>: 2                           [2단 치수선을 선택]
Distance or RETURN to Pick point: 19500        [알고 있는 거리는 직접 입력함]
Dimension text <19.500>:               [그대로 사용함, 고치고 싶다면 원한는 text입력]
Distance or RETURN to Pick point/eXit:         [DH명령 끝냄]
