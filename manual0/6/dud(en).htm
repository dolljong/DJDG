Up and Down dimension text. 
 
  File name : DUD.LSP


  AutoCAD has enviroment variable dimension DIMTVP. 
We can set position of dim text up and down by changing DIMTVP.
But It is bother to change DIMTVP whenever draw dimension.
So you can change position of dim text by using this program after you draw all dimensions.

 
   Example
  Select entity:        [ Select dimension entitis aleady drawn]
  Up/<Down>: D [Enter 'D' to move text to down]
                   U        [            "             Up]
  Select entity:       [Enter  to finish]
