경사진면을 따라 구간 Array를 해줍니다.

 
  Program name : ARSLOP.LSP


슬라브 단면 배근도에서 배력근을 c.t.c 120로 할 때 x방향 거리가 120이기 때문에 AutoCAD의 
array명령을 이용해서 그리려면 좀 불편합니다. 이 프로그램은 경사선을 따라 array할 때 x방향 
또는 y방향의 간격을 지정하여 array할 수 있습니다.

 
     사용예               
Select objects: {p1}1 found     [array할 엔티티선택]
Select objects:                        [선택 끝냄]
Pick start point: {p2}          [시작점 찍음]
Pick end point: {p3}                    [끝점 찍음]
Array distance: 600                    [array간격 입력]
X-direction/Y-direction: x     [array방향 입력]
Select line/2P: {p4}                    [array경사 입력, 
                                  경사를 가진 line을 선택하는 방법과
                                  2p?를 입력하고 두 점을 찍어서 정하는 방법이 있슴]
10@600.0=6000.0                 [프로그램이 array간격과 개수, 총길이 등을 표시해줌]
