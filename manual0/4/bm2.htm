철근마킹의 보조선을 그려줍니다.
 
   Program name : BM2.LSP


  아래의 그림과 같이 여러개의 철근을 묶어서 하나의 마킹으로 처리할 때 보조선을 그려줍니다. 
    ◆ 주의사항
      ▷ 선택된 철근선들은 호이거나 직선이어야 합니다.
      ▷ 선택된 철근중에 보조선과의 교점이 존재하지 않는 철근이 있으면 오류가 발생합니다.

 
     사용예 
     Select objects: f         [한꺼번에 여러개의 철근을 선택하기 위해 fence option사용]
     first fence point: {p1}            [fence의 첫째점 입력]
     Undo/<Endpoint of line>: {p2}      [fence의 두번째점 입력]
     Undo/<Endpoint of line>:  [fence option끝냄]
     Select objects:                   [object선택을 끝내기 위해 Enter입력]
     4 Entity Found                     [선택된 entity갯수 표시]
     Pick start point: nea to {p3}      [보조선의 시작점 찍어줌]
     Pick end point: nea to {p4}                [보조선의 끝점 찍어줌]
