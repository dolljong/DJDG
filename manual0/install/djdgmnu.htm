AutoCAD 2000에서 [다정다감]메뉴 띄우기

[다정다감]을 공개하고 종종 받아온 질문이 바로 '[다정다감]을 설치했는데도 CAD에 아무런 변화가 없어요~'라는 것이었습니다. djdg2k0102.exe를 실행으로 설치한 파일들은 [다정다감]실행에 필요한 파일들을 적절한 디렉토리를 만들어 넣은 것으고 실제로 AutoCAD 2000에 [다정다감]을 탑재하는 것은 이번 장에서 설명하는 것을 보셔야 합니다. 종종 급한 성격을 가지신 분들이 setup.exe의 실행이 끝나자마자 설치매뉴얼의 나머지 부분은 읽어 보시지 않고 AutoCAD로 바로 들어가보고 아무변화도 없는 것을 확인하고 '[다정다감] 순 사기다'라고 하시는 것 같습니다.

앞에서도 언급했지만 이번 버전은 앞의 버전들과 설치방법이 약간 다르므로 기존에 [다정다감]을 사용하셨던 분들도 끝까지 주의깊게 읽어주시기 바랍니다.

  • djdg.dwt선택하기

    AutoCAD 2000을 실행시키면 위와 같은 화면이 나올 것입니다.

    (물론 옵션을 꺼두셨다면 위와 같은 box가 뜨지 않을 것입니다. tools|options|system에서 show startup dialog를 체크해두면 매번 AutoCAD 2000을 실행시킬 때마다 위의 box가 뜹니다.) [다정다감]이 제공하는 기능들을 보다 더 정확하게 사용하려면 될 수 있으면 [다정다감]이 제공하는 [다정다감]용 template파일을 사용하실 것을 권장합니다. [다정다감]프로그램이 사용하는 font style이라든가 글자크기 치수선 옵션등을 일반적인 토목구조도면에 맞게 맞춰두었습니다. 위의 그림과같이 버튼을 누르면 template 디렉토리에 있는 사용가능한 template(*.dwt)파일 목록이 나오는데 이중에서 [다정다감]이 제공한 djdg.dwg를 선택합니다. 매번 djdg.dwt를 선택하기 불편하시다면 원본 acad.dwt를 acado.dwt로 이름을 바꿔두고 djdg.dwt를 acad.dwt로 바꿔두면 캐드가 실행될 때마다 djdg.dwt를 기반드로 모든 환경들이 설정됩니다.

    djdg.mns를 선택하고 를 누르면 아래와 같이 다시 Menu customization box로 돌아갑니다.

    를 눌러 [다정다감] 메뉴그룹을 load합니다. 주의할 것 한가지가 있다면 꼭 꺼두시라는 것입니다. 만일 이 옵션이 켜있다면 [다정다감] 메뉴하나만 달랑 나타나게 됩입니다. 메뉴그룹이 정상적으로 load되었다면 아래와 같이 ACAD아래에 DJDG라는 메뉴그룹이 표시되고 [다정다감] tool bar들이 AutoCAD창 여기저기에 뜹니다.

    Menu Group에 DJDG가 추가되어있는 것을 알 수 있습니다. DJDG를 선택하면 아래와 같이 DJDG 메뉴그룹에 들어있는 popup메뉴항이 나타납니다.

    [다정다감]의 popup메뉴는 DJDG하나입니다. 왼쪽창의 DJDG popup메뉴를 선택하여 오른쪽창의 Menu Bar에 추가하는 형태입니다. 오늘쪽 창에서 마음에 드는 위치를 선택하고(저는 Help앞에 넣기로 했습니다) 를 누르면 DJDG메뉴가 Help앞에 위치하게 됩니다. 를 눌러 메뉴띄우기를 마칩니다.
