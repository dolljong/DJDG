Excel의 데이터를 CAD의 Pline으로 Xls2Pline  

Excel로 작업된 데이터를 CAD의 그림으로 만들어야 되는 경우가 종종 있습니다. BMD를 그려야 되는 경우도 있을 것이고 그래프를 그린다든가, 종단을 그려야 되는 경우도 있을 것입니다. [다정다감] for CAD에는 xypline이라는 기능이 있지만 데이터를 text형태로 작성해야 되기 때문에 Excel로 작업한 데이터를 xls2pline을 이용하여 pline으로 그리기 위해서는 약간의 번거로운 Edit작업이 필요했습니다. 이제 Excel에서 작업한 경우라면 xls2pline으로 간단하게 pline을 그릴 수 있습니다.

기능
  • Excel의 x,y값을 이용하여 CAD에 Pline또는 line그리기
  • 한개의 x값에 2개 이상의 y값 그리기 가능
  • x,y scale을 다르게 그리기 가능
사용법

위 그림과 같이 x,y좌표가 들어있는 셀을 선택하고 [다정다감] toolbar의 xls2pline버튼 클릭합니다. 만일 [다정다감] toolbar가 보이지 않는다면 이 도움말의 djdg.xla설치하기 를 참조하시기 바랍니다.

xls2dwg 버튼을 클릭하면 낯익은 다이얼로그 박스가 하나 뜹니다. 어디서 많이 본듯한 다이얼로그 박스라고 생각되지 않으십니까? 캐드의 insert명령 다이얼로그 박스와 유사합니다.

사용자의 취향에 알맞게 옵션을 선택한 후에 ok버튼을 누르면 아래와 같이 캐드에 pline이 그려집니다.



옵션

그럼 옵션에 대해서 간단하게 설명드리겠습니다. 캐드를 사용해보신 분들은 대충은 이해하시리라 생각됩니다. 다이얼로그박스는 크게 세부분으로 나누어 생각할 수 있습니다. insert point와 scale, Explode 이 그것입니다.

Insert point를 직접 CAD의 화면에석 찍을 때 사용됩니다.
첫번째 점을 Insert point로 사용할 때 선택합니다.
x,y Scale이 일정할 때 사용합니다. 이 체크박스를 선택하면 x스케일이 y스케일에 똑같이 적용됩니다.
pline으로 그리지 않고 line으로 그려줍니다.

사용예

여러개의 y값 그리기
한개의 x그룹에 여러개의 y그룹 그리기도 가능합니다. y그룹이 여러개인 그래프나 계획고, 지반고등 여러개의 y값이 종단곡선 같은 것을 그릴 때 유용할것입니다.


x또는 y스케일 조정하기
종단을 그리는 경우에는 X스케일와 y스케일을 다르게 하는 경우가 있습니다. 이때 Excel의 데이터를 고쳐서 그려도 되지만 Xls2pline의 Scale옵션을 사용하면 원본데이터 건드리지 않고도 x스케일과 y스케일이 다른 pline을 그릴 수 있습니다.
