Draw circle or box on existing text
 
  Program name : TM.LSP


  Draw mark on text writen aleady. Mark types are 4 type, circle, box, ellipse, track .
  When you choose box, ellipse, track option, You can choose a option "Fit" that program determine automaticaly size of mark. 
If you don't choose option "Fit", program ask size of mark.

 
 Example-1 Circle
Select objects: Other corner: 1 found                   [Select texts]
Select objects:   Circle/Box/Ellipse/Obang: c [Choose circle type]
Pick center point:                                      [Pick center point]
Radius:                                         [Pick a point to determine radius]

 
 Example-2 BOX
Select objects: 1 found                                 [Select texts]
Select objects:   Circle/Box/Ellipse/Obang: b [Choose box type]
<Box width>/Fit:  Second point:                 [Pick two points to determine size of box, You can enter numbers also]
Box Height:  Second point:                      [Pick two points, You can enter numbers also]

Box Height:  Second point:                      [Pick two points to determine size of Ellipse, You can enter numbers also]
